# 对话型 AI 使用场景

老张API 为各种对话型 AI 工具提供稳定可靠的国内 ChatGPT API 访问服务。

## 支持的对话型 AI 工具

### Cherry Studio
国内 ChatGPT 使用配置指南

#### 配置步骤
1. 在 Cherry Studio 设置中找到 API 配置
2. 将 API 端点设置为：`https://api.laozhang.ai/v1`
3. 输入您的 老张API 密钥
4. 选择需要的模型（如 GPT-4、Claude 等）

### Chatbox AI
#### 配置说明
- API 地址：`https://api.laozhang.ai/v1`
- 模型选择：支持所有主流模型
- 流式响应：完美支持

### Open WebUI
国内 ChatGPT 部署配置指南

#### 部署配置
```yaml
# docker-compose.yml 配置
environment:
  - OPENAI_API_BASE=https://api.laozhang.ai/v1
  - OPENAI_API_KEY=您的API密钥
```

### ChatGPT Next Web
国内部署配置指南

#### 环境变量配置
```bash
# .env 文件配置
OPENAI_API_BASE_URL=https://api.laozhang.ai/v1
OPENAI_API_KEY=您的API密钥
```

## 通用配置建议

- **API 端点**: `https://api.laozhang.ai/v1`
- **模型兼容**: 完全兼容 OpenAI API 格式
- **网络优化**: 国内服务器，低延迟访问
- **支付方式**: 支持支付宝等多种支付方式
