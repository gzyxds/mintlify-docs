## 概览

本页将带你在 5 分钟内完成以下目标：

- 注册 必定AI 账号并完成充值
- 创建并获取 API 密钥
- 完成首次 API 调用

只需简单三步，即可开始使用 必定AI 访问各种 AI 模型。

## 步骤 1：账户准备

<AccordionGroup>
  <Accordion icon="user-plus" title="注册账号">
    访问 必定AI官网注册账号：
    
    1. 使用邮箱注册（推荐高校、企业邮箱注册）
    2. 验证邮箱地址
    3. 登录控制台
    
    新用户注册即可获得300万Token免费测试额度，约可调用4o-mini模型进行充分测试。
  </Accordion>

  <Accordion icon="credit-card" title="账户充值">
    在控制台进行充值：
    
    1. 点击"充值"菜单
    2. 选择充值金额（最低 5 美金起，100 美金可加赠 15%）
    3. 完成支付（支持支付宝、微信）
    
    新用户首次充值可获得额外赠送，充值后余额立即到账，首充加赠额度需要加工作人员微信后为你人工加赠（微信：8765058）。
  </Accordion>
</AccordionGroup>

## 步骤 2：创建 API 密钥

<AccordionGroup>
  <Accordion icon="key" title="生成密钥">
    在后台导航点击"令牌"栏目，https://api.apiyi.com/token
    
    1. 可以复制【默认令牌】直接使用（右侧有个复制图标）
    2. 或者新创建令牌：点击右上侧"新增"按钮，为密钥命名（如：test-key），点击确认创建，即拥有新的令牌。
    
    <Tip>
      安全建议：
      - 不要将密钥写入仓库或前端代码
      - 使用环境变量存储
      - 一旦泄露，立即在控制台重置密钥
    </Tip>
  </Accordion>
</AccordionGroup>

## 步骤 3：开始调用

### 获取接入信息

调用时只需要记住 3 个核心参数：

```bash
# API 基础地址
base_url: https://api.apiyi.com/v1

# API 密钥（在控制台获取）
api_key: YOUR_API_KEY

# 模型名称（可按需切换）
model: gpt-4.1-mini      # 通用对话
```

### 测试调用

使用 curl 快速测试：

```bash
curl https://api.apiyi.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4.1-mini",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

### 代码示例

<Tabs>
<TabItem label="Python">
```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.apiyi.com/v1"
)

response = client.chat.completions.create(
    model="gpt-4.1-mini",
    messages=[
        {"role": "user", "content": "Hello!"}
    ]
)

print(response.choices[0].message.content)
```
</TabItem>

<TabItem label="Node.js">
```javascript
import OpenAI from 'openai';

const openai = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://api.apiyi.com/v1',
});

const completion = await openai.chat.completions.create({
  messages: [{ role: 'user', content: 'Hello!' }],
  model: 'gpt-4.1-mini',
});

console.log(completion.choices[0].message.content);
```
</TabItem>

<TabItem label="Java">
```java
import com.theokanning.openai.service.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.completion.chat.ChatMessageRole;

import java.util.Arrays;

public class Main {
    public static void main(String[] args) {
        OpenAiService service = new OpenAiService("YOUR_API_KEY", "https://api.apiyi.com/v1");
        
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("gpt-4.1-mini")
                .messages(Arrays.asList(
                    new ChatMessage(ChatMessageRole.USER.value(), "Hello!")
                ))
                .build();
                
        System.out.println(service.createChatCompletion(request).getChoices().get(0).getMessage().getContent());
    }
}
```
</TabItem>
</Tabs>

<Tip>
如果你已经在使用 OpenAI 官方 SDK，通常只需要：

1. 将 `api_key` 换成 必定AI 的 Key  
2. 将 `base_url` 改为 `https://api.apiyi.com/v1`  

即可在不改业务逻辑的情况下完成迁移。
</Tip>
