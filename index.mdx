BuidAPI是获得微软 Azure、AWS、GCP 官方授权的合规模型聚合平台。公司依托正规云服务额度和专业技术团队，长期为众多知名应用提供高并发、稳定、可信赖的 AI API 服务。

平台基于统一的 OpenAI API 标准，聚合了 OpenAI 官方全系列模型与 Claude、Gemini、DeepSeek、阿里 Qwen 等主流大模型。开发者只需更改模型参数即可一键切换，无缝对接多家厂商，极大降低接入与运维成本。

### 多端点支持说明

BuidAI 会同时链接 OpenAI 官方和微软 Azure OpenAI 两个端点，并随机路由请求至 Azure 进行内容审核。相比 OpenAI 官方端，Azure OpenAI 自带自动内容过滤，所以响应速度可能略慢。

<Warning>
请注意合规：不要请求暴力、政治、色情等敏感内容。若账号频繁触发内容过滤，我们将进行统计并与您联系核实。
</Warning>
